# Zymovo - Premium Static Website

This is a static website for Zymovo, designed to convey a high-end, large-scale, premium, and visually impressive brand identity for a synthetic biology laboratory / advanced biotech company.

## Technical Details

*   **Pure Static Site:** Built with HTML, CSS, and JavaScript.
*   **No Frameworks:** No external JavaScript or CSS frameworks used.
*   **No Build Step:** The site is ready for deployment as is.
*   **Multi-page:**
    *   `index.html` (Homepage)
    *   `platform.html` (Our Platform)
    *   `research.html` (Research & Innovation)
    *   `about.html` (About Us)
    *   `contact.html` (Contact Us)
*   **Deployment:** Designed for direct deployment to platforms like Cloudflare Pages.

## Content & Design

*   **Tone:** Premium, confident, scientific, elegant.
*   **Visuals:** Utilizes provided image assets (located in `zymovoimages/`) and incorporates a dark, sophisticated aesthetic.
*   **Contact Information:**
    *   Contact Person: Yulin D
    *   Email: info@Zymovo.com

## Setup & Deployment

1.  Clone this repository.
2.  The `index.html` file is the entry point.
3.  Deploy the entire directory to a static site hosting service (e.g., Cloudflare Pages, Netlify).
